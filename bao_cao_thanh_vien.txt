Chi tiết công việc từng thành viên

THÀNH VIÊN 1: [Tên]
- Thiết kế cơ sở dữ liệu (database/schema.sql)
- Xây dựng module Admin:
  + Trang dashboard tổng quan (admin/dashboard.php)
  + Quản lý người dùng (admin/users.php)
  + Quản lý cửa hàng (admin/shops.php)
  + Quản lý danh mục (admin/categories.php)
  + Quản lý voucher (admin/vouchers.php)
  + Cấu hình hệ thống (admin/settings.php)
  + Báo cáo tài chính (admin/finance.php)
  + Xử lý hỗ trợ (admin/support.php)

THÀNH VIÊN 2: [Tên]
- Xây dựng module Khách hàng:
  + Trang chủ khách hàng (customer/index.php)
  + Xem danh sách cửa hàng (customer/shops.php)
  + Chi tiết cửa hàng (customer/shop_detail.php)
  + Giỏ hàng (customer/cart.php)
  + Thanh toán (customer/checkout.php)
  + Quản lý đơn hàng (customer/orders.php, order_detail.php)
  + Theo dõi đơn hàng (customer/track_order.php)
  + Đánh giá (customer/review.php, reviews.php)
  + Hồ sơ cá nhân (customer/profile.php)
- Xây dựng module xác thực (auth/login.php, register.php, logout.php)

THÀNH VIÊN 3: [Tên]
- Xây dựng module Người bán:
  + Trang dashboard (seller/dashboard.php)
  + Đăng ký cửa hàng (seller/register_shop.php)
  + Quản lý sản phẩm (seller/products.php)
  + Quản lý combo (seller/combos.php)
  + Quản lý đơn hàng (seller/orders.php, order_detail.php)
  + Tạo khuyến mãi (seller/promotions.php)
  + Xem doanh thu (seller/revenue.php)
  + Phản hồi đánh giá (seller/reviews.php)
  + Nhắn tin khách hàng (seller/chat_customer.php)

THÀNH VIÊN 4: [Tên]
- Xây dựng module Shipper:
  + Trang dashboard (shipper/dashboard.php)
  + Đơn hàng của tôi (shipper/my_orders.php)
  + Cập nhật trạng thái (shipper/update_status.php)
  + Trạng thái sẵn sàng (shipper/available.php)
  + Xem thu nhập (shipper/earnings.php)
  + Bản đồ đơn hàng (shipper/order_map.php)
  + Nhắn tin khách hàng (shipper/chat_customer.php)
- Xây dựng API:
  + API sản phẩm (api/products.php)
  + API giỏ hàng (api/add_to_cart.php)
  + API tính phí ship (api/calculate_shipping.php)
  + API vị trí (api/save_location.php, shipper_location.php)


Minh chứng đóng góp trên GitHub

[Chèn ảnh chụp màn hình GitHub Contributors tại đây]

Hướng dẫn lấy ảnh:
1. Truy cập repository GitHub của dự án
2. Click vào tab "Insights" → "Contributors"
3. Chụp màn hình biểu đồ đóng góp của các thành viên
4. Hoặc vào trang chính repo, cuộn xuống phần "Contributors" bên phải


Nhận xét chung

- Tất cả thành viên đã hoàn thành công việc được giao đúng tiến độ
- Các module được tích hợp và hoạt động ổn định
- Code được quản lý trên GitHub với các commit rõ ràng
- Các thành viên hỗ trợ lẫn nhau trong quá trình phát triển
